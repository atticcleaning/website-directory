# Data Sources

## US Postal Code Data (US.txt)
- Source: GeoNames.org (https://www.geonames.org/)
- License: Creative Commons Attribution 4.0 (CC-BY 4.0)
- Downloaded: 2026-02-13
- Format: TSV (tab-separated), 12 columns, no headers
- Records: ~41,500 rows (raw file includes US territories; seed script filters to 50 states + DC only, yielding ~41,000 unique zip codes)
